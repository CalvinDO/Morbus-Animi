Geometrie

Pivot/Origin

Allgemein befindet sich der Pivot von Objekten an der Stelle, an der sie entweder montiert sind, oder darauf stehen.

Bei breiteren Objekten am besten in der Mitte vom Objekt auf der Bodenebene, so dass sie einfach auf die gleiche Höhe des Bodens gesetzt werden können, oder, wenn sich ein anderer logischer Schluss ergibt, auf dieses.

Bei Räumen/ Tunneln bitte den Pivot auf Welt-0 der Höhen-Achse setzen.

Bewegliche Objekte haben ihren Pivot an der Stelle, um die sie sich im Spiel drehen sollen.

Objekte werden nicht voneinander getrennt, wenn es nicht notwendig ist, und viele Objekte erzeugt. Beispiel: Blume (die nicht zerissen werden soll) -> Blätter existieren nicht als einzelne Objekte.

Möbel o.Ä. können vom Raum getrennt werden, da sie eventuell wiederverwendet werden und es im Nachhinein Arbeit ist, sie zu trennen.

Rotation/Orientierung der Objekte:
Das lokale Koordinatensystem eines Objektes sollte möglichst entlang seiner tatsächlichen Ausrichtung angebracht sein.
In Blender:

  • Die Z-Achse zeigt entlang des Objektes nach OBEN!
  • Die Y-Achse zeigt entlang des Objektes nach HINTEN!
  • Die X-Achse zeigt entlang des Objektes nach LINKS!

Skalierungen

Rechtshändiges zu Linkshändiges Koordinatensystem - Blender zu Unity

Blender-Export für alles AUßER Character:
  1. Ctrl + A -> Apply Rotation & Scale
  2. file -> Export -> Unity FBX
  3. Selected Objects only = an, wenn es Objekte gibt die nicht exportiert werden sollen
  4. Export Unity FBX
  5. fertig :)
Blender-Export für Character:
  • Ctrl + A -> Apply Rotation & Scale
  • Character Mesh auswählen, Shift Select Rig dazu auswählen
  • file -> Export -> FBX
  • Include
    • Selected Objects only = an
    • Bei den zu exportierenden Objekttypen nur Mesh, Armature und Empty auswählen Export Unity FBX
  • Transform
    • Scale = 1.00
    • Apply Scalings = FBX All (viele benutzen All Local, doch das gibt bei uns anscheinend Probleme)
    • Forward = Z Forward
    • Up = Y Up
  • Armature -> Add Leaf Bones AUS! Gibt schlechte Performance! Ausnahme: Dynamic Hair Rigging, etc. (Bitte vorher andere aus dem Team fragen)
  • Bake Animation -> Bake Animation an.
    • Key all Bones = an
    • NLA Strips = aus
    • All Actions = an (Wenn es Probleme gibt, dass Animation (Aktionen) exportiert werden, die gar nicht mehr wirklich vorhanden sind, dann lieber AUS schalten)
    • Force Start/End Keying = aus
  • Export FBX :) fertig :)
